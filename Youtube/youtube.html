Youtube Channel

Welcome to my Youtube Channel! Here, I share videos showcasing my journey through robotics and programming. Right now, I mainly make videos on Raspberry Pi and Arduino for robotics, as well as Python (maybe some Java in the future) for programming. My goal is to provide the start of a project for you. I’ll make small prototypes that you can then develop into projects. You’ll find detailed project walkthroughs, code explanations, and demonstrations that make it easy to follow along. One of my recent videos is the clapper project, where I demonstrate how to build a sound-activated led using the KY-037 sound detection module. I’ve provided detailed steps for the wiring and walked you through the code so you understand what is happening. The concept can easily be scaled to room lights. Whether you’re a beginner curious about robotics or an experienced maker looking for fresh ideas, my videos aim to make complex concepts approachable and fun.
